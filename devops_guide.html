Networking Foundations Refresher

‚è±Ô∏è Duration: 2-3 weeks üìö Level: Foundation üéØ Prerequisites: Basic IT knowledge

‚úÖ Your Progress

üìö Core Networking Concepts

  • TCP/IP Stack: Deep dive into OSI vs TCP/IP models, packet encapsulation, headers analysis
  • Subnetting Mastery: CIDR notation, VLSM, IPv4 address planning, subnet calculations
  • DNS & DHCP: Name resolution process, DNS record types (A, AAAA, CNAME, MX, TXT), DHCP lease management
  • NAT & PAT: Network Address Translation, Port Address Translation, NAT types (static, dynamic, PAT)
  • VLANs & Switching: Virtual LANs, 802.1Q tagging, trunk ports, inter-VLAN routing
  • Routing Protocols: Static routing, RIP, EIGRP, OSPF concepts, BGP basics for cloud

‚òÅÔ∏è Cloud Networking Bridge Critical for DevOps

  • VPC Fundamentals: AWS VPC, Azure VNet, GCP VPC - understand how traditional networks map to cloud
  • Subnets & Routing: Public vs private subnets, route tables, internet gateways, NAT gateways
  • Security Groups & NACLs: Stateful vs stateless firewalls, inbound/outbound rules
  • Load Balancers: Layer 4 vs Layer 7, Application Load Balancer (ALB), Network Load Balancer (NLB)
  • VPN & Direct Connect: Site-to-site VPN, AWS Direct Connect, Azure ExpressRoute, hybrid connectivity
  • DNS in Cloud: Route 53, Azure DNS, Cloud DNS - routing policies, health checks

‚úÖ Checkpoint Exercise

Hybrid Network Design Project:

Design and document a complete hybrid cloud network architecture:

  • Create AWS VPC with public/private subnets across 2 AZs
  • Configure security groups for web, app, and database tiers
  • Setup NAT gateway for private subnet internet access
  • Design VPN connection to simulate on-premises network
  • Document IP addressing scheme and routing tables
  • Draw network diagram using draw.io or Lucidchart

üìÖ Weekly Learning Timeline

W1
Week 1: TCP/IP & Subnetting Deep Dive

Review OSI model layers, practice subnetting with online calculators, setup GNS3 or Packet Tracer for labs, complete 50+ subnetting exercises

W2
Week 2: DNS, NAT, VLANs Configuration

Configure DNS servers, implement NAT/PAT on routers, create VLAN scenarios, practice inter-VLAN routing, troubleshoot connectivity issues

W3
Week 3: Cloud Networking Hands-on

Create AWS free tier account, build VPC from scratch, configure security groups and NACLs, deploy EC2 instances in public/private subnets, setup load balancer

Linux Fundamentals

‚è±Ô∏è Duration: 3-4 weeks üìö Level: Essential üéØ Daily Practice Required

‚úÖ Your Progress

üîß Essential Linux Commands Daily Use

  • File Management: ls, cd, pwd, cp, mv, rm, mkdir, rmdir, touch, find, locate
  • Text Processing: cat, less, head, tail, grep, awk, sed, cut, sort, uniq, wc
  • Permissions: chmod, chown, chgrp, umask, ACLs (setfacl, getfacl), sudo configuration
  • Process Management: ps, top, htop, kill, killall, pkill, nice, renice, bg, fg, jobs
  • Networking: ip, ifconfig, netstat, ss, ping, traceroute, tcpdump, nc, curl, wget
  • Package Management: apt/apt-get (Debian/Ubuntu), yum/dnf (RHEL/CentOS), managing repositories
  • Archive & Compression: tar, gzip, bzip2, zip, unzip, rsync for backups

‚öôÔ∏è System Administration Critical Skills

  • SSH Configuration: Key-based authentication, ssh-agent, config file (~/.ssh/config), tunneling, port forwarding
  • Systemd Services: Understanding unit files, systemctl commands (start, stop, enable, disable), creating custom services
  • Cron Jobs & Scheduling: Crontab syntax, scheduling tasks, anacron, systemd timers as alternative
  • Log Management: /var/log directory, syslog, journalctl commands, logrotate configuration
  • Storage Management: LVM (Logical Volume Manager), mounting filesystems, /etc/fstab, disk utilities (fdisk, parted, df, du)
  • User Management: useradd, usermod, userdel, group management, /etc/passwd, /etc/shadow

‚úÖ Checkpoint Exercise

Deploy Production Web Server:

  • Install and configure Nginx or Apache web server
  • Setup SSH key-based authentication (disable password auth)
  • Create systemd service for a custom application
  • Configure automated backup cron job to run daily
  • Implement log rotation for application logs
  • Setup firewall rules using ufw or iptables
